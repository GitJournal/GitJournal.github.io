Version 1.70

2020-09-16

Features

  • Implement Basic Search highlighting #14

  • Implement basic note selection

  • Single Journal entry Per Day

    Some users prefer all the journal entries for a single day be in the same file. This matches with how Obsidian and Foam handle the daily journal - #243

Improvements

  • Purchase Screen - Make it prettier for smaller screen sizes

  • Make all strings in the app translatable

  • Configuring the default Image settings is now part of `GitJournal Basic`

Bug Fixes

  • Do not show Tag Editing screen if the Note cannot have metadata

  • Zettelkasten FileName Mode - Add leading zero to minutes < 10 - #247

  • Dark Mode: Fix blockquote background color
