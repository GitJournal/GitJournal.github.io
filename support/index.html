FAQ

How do you plan to sustain this project?

Sustaining an Open Source Project can be challenging in the long term. The idea with GitJournal is to be a 100% Open Source Project - No Open Core model. Here are our current ideas on how to sustain this in the long term -

  1. Paid Pro Version - Some Advanced features could require payment. The source code for these features would still be available here, and users are free to install a version with all the features unlocked - but the versions on Google Play Store and iOS would require payment.

  2. Encrypted Git Hosting - Our other idea was to provide paid end-to-end encrypted Git Hosting which could be used to store the notes.

We’re open to suggestions on how to sustain GitJournal. Send us an email with your thoughts at hello@gitjournal.io or comment publicly on Github

Does GitJournal work with SSH on a custom port?

Yes. Please specify the full url with the ssh protocol. Eg - ssh://user@host.com:PORT/path/to/repo/journal.git
